# JavaScript Learning Repository

> **Note:** This repository was generated by AI as a learning exercise.

A comprehensive JavaScript learning curriculum with hands-on exercises covering fundamentals through advanced concepts.

## Quick Start

```bash
# Run any exercise
node Exercises/Phase1/1-variables.js
```

## Learning Path

| Phase | Topic | Est. Time |
|-------|-------|-----------|
| 1 | Fundamentals (variables, functions, arrays) | 4-5 hrs |
| 2 | Intermediate (closures, DOM, async) | 10-12 hrs |
| 3 | Working with JavaScript (Fetch, modules, npm) | 6-8 hrs |
| 4 | Advanced (prototypes, generators, regex) | 8-10 hrs |
| 5 | Ecosystem (Node.js, TypeScript, React) | 16-20 hrs |
| 6 | Professional (patterns, performance, security) | 6-8 hrs |

**Total: ~80-100 hours**
