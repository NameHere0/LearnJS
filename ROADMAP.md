# JavaScript Learning Roadmap

Estimated total time: **80-100 hours**

Each phase includes exercises in `Exercises/PhaseX/` with reference markdown files.

---

## Phase 1: Fundamentals
**Estimated time: 4-5 hours**

- [ ] Variables, Data Types, and Operators
- [ ] Control Flow (if/else, switch, loops)
- [ ] Functions (declarations, expressions, arrow functions)
- [ ] Arrays and Objects
- [ ] String Methods and Manipulation

**Exercises:** 5 exercises (~3.5 hours) + Project time

---

## Phase 2: Intermediate Core
**Estimated time: 10-12 hours**

- [ ] Scope, Closures, and Hoisting
- [ ] DOM Manipulation
- [ ] Events and Event Handling
- [ ] ES6+ Features (let/const, destructuring, spread, template literals)
- [ ] Array Methods (map, filter, reduce)
- [ ] Error Handling (try/catch)
- [ ] Asynchronous JavaScript (callbacks, promises, async/await)

**Exercises:** 7 exercises (~8 hours) + Project time

---

## Phase 3: Working with JavaScript
**Estimated time: 6-8 hours**

- [ ] Fetch API and HTTP Requests
- [ ] Working with JSON
- [ ] Local Storage and Cookies
- [ ] Module Systems (ES Modules, CommonJS)
- [ ] npm and Package Management
- [ ] Build Tools (Vite, Webpack basics)

**Exercises:** 4 exercises (~4 hours) + Project time

---

## Phase 4: Advanced Concepts
**Estimated time: 8-10 hours**

- [ ] Prototypes and Inheritance
- [ ] 'this' keyword and Context
- [ ] Generators and Iterators
- [ ] Proxy and Reflect
- [ ] Symbol and Well-Known Symbols
- [ ] Regex (Regular Expressions)
- [ ] Web APIs (Canvas, WebRTC, etc.)

**Exercises:** 5 exercises (~5.5 hours) + Project time

---

## Phase 5: Ecosystem & Frameworks
**Estimated time: 16-20 hours**

- [ ] Node.js Basics
- [ ] TypeScript Fundamentals
- [ ] React, Vue, or Angular (choose one)
- [ ] State Management
- [ ] Testing (Jest, Vitest)
- [ ] Next.js or similar frameworks

**Exercises:** 4 exercises (~7.5 hours) + Project time

---

## Phase 6: Professional Skills
**Estimated time: 6-8 hours**

- [ ] Design Patterns
- [ ] Performance Optimization
- [ ] Security Best Practices
- [ ] CI/CD and Deployment
- [ ] Code Organization and Architecture

**Exercises:** 3 exercises (~4.5 hours) + Project time

---

## Recommended Learning Path

1. **Week 1-2:** Phase 1 - Complete fundamentals
2. **Week 3-4:** Phase 2 - Master intermediate concepts
3. **Week 5-6:** Phase 3 - Learn to work with JavaScript ecosystem
4. **Week 7-8:** Phase 4 - Deep dive into advanced topics
5. **Week 9-12:** Phase 5 - Learn a framework (React recommended)
6. **Week 13-14:** Phase 6 - Professional development skills

## Tips for Success

1. **Practice daily:** 1-2 hours of focused coding
2. **Read reference files:** Each exercise has a corresponding markdown guide
3. **Don't skip exercises:** They reinforce learning
4. **Build projects:** Apply skills after each phase
5. **Review regularly:** Revisit completed phases monthly
6. **Join communities:** Stack Overflow, Reddit, Discord for help
